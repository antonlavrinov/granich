Толковые инструменты

Microsoft To Do

Для ведения дел

Microsoft To Do

Фрилансеру не нужна CRM. У вас не сотни клиентов, а 2-3. Все задачи легко отмечать в заметках.

От создателей культового Wunderlist. В 2015 Вандеров выкупил Микрософт. Теперь ребята делают прогу для ведения задач под брендом Билла Гейтса. По сути, To Do это улучшенная, актуальная версия Wunderlist. Идеальна для ведения списков задач. Есть на всех платформах, можно открывать списки дел команде, назначать сроки, делать оповещения.

Miro

Для командной работы

Realtime Board (Miro)

Бесплатные доски для командной работы. Майндмэпы, канбан, заметки, загрузка картинок и чёрте что еще, жирная штука. P.S. у них ребрендинг. Теперь это Miro.

Notion

Для ведения дел

Notion

Великолепный инструмент для больших объемов информации. Например, мы в онлайн-школе Granich используем его чтобы сохранять различные чек-листы, ответы на частые вопросы, скрипты. И самое классное, у Notion бесплатный персональный план. Для личных целей можно пользоваться абсолютно бесплатно. Мы любим использовать Notion в связке с Microsoft To Do. В Notion указывать всю объёмную инфу которая должна долго храниться, а в Microsoft To Do контролировать операционные задачи.

Google Документы

Для редактирования текста

Google Документы

Во-первых, инструмент бесплатный, во-вторых, у него есть потрясающая функция совместного онлайн-редактирования и комментирования. Вы можете добавить любого человека с любой почтой и дать ему, например, право только комментировать документ. Таким способом можно быстро обсуждать договор на проект или техническое задание с заказчиком, отвечая на его вопросы по конкретным пунктам прямо в документе. А можно например добавить нескольких людей для редактирования документа. Так мы делаем когда составляем новые тексты на экзамены к курсу Осознанный Графдизайн.
